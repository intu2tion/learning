인터넷과 웹?

[유튜브 참고] 생활코딩 - web1 원시웹 유튜브

internet vs web?
인터넷이 도로라면 웹은 도로위를 달리는 자동차 한대이다.

1960년: 인터넷의 등장
1990년: 웹의 출현

인터넷을 여는 열쇠: 서버와 클라이언트

(Web Browser)요청하는 컴퓨터 client 와
(Web Server)응답하는 컴퓨터 server

게임 클라이언트(사용자가 게임을 하는 컴퓨터) 와 게임사가 갖고있는 서버컴퓨터.
웹 서버라는 것은 추상적이고 낯설고 막막하게 느껴지겠지만, 익숙해지면 쉽게 느껴집니다.
써보고 말해보고 계속 보다보면 익숙해집니다.
익숙해지는 것중에 하나는 이해하는 것.

자신의 컴퓨터에 웹 서버를 깔거나, 대행해주는 업체에게 맡기는 방법이 있다(hosting)
웹 서버를 까는 것은 어렵지만 인터넷과 관련된 여러 개념을 배울수 있습니다.
웹 호스팅을 맡기는 것은 쉽지만 배울 건 적다.

  • 생각의 재료: 웹호스팅으로 쉽게 해보고, 웹서버를 통해 원리를 파악하는 방향으로 해볼 것.
