반응형 디자인(Responsive Web)

웹의 해상도, 레이아웃 등이 스크린 환경에 따라 반응하여 변환되는 웹 페이지.
(화면의 크기에 따라 웹 페이지의 각 요소들이 반응해서 동작하게 된다.)

미디어 쿼리: 반응형 디자인을 CSS를 통해 구현

screen width < 800 px ---> @media(max-width:800px){}
화면 크기가 800px 보다 작다면,
즉, 화면의 크기가 최대 800px 일 때 { } 실행
모바일 화면 비중이 높다면 기준을 MIN으로, PC화면 비중이 높다면 기준을 MAX로.

responsive(800px이하에서만 구현) 화면 크기를 바꾸면 반응함.
