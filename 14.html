Object ?

Object(객체) = 서로 연관된 함수와 변수를 같은 이름으로 그루핑해서 잘 정리정돈하기 위한 도구.

객체에 포함된 함수를 method 라고 한다.
배열: var coworkers = ['intu2tion', 'hijy'];
객체: var coworkers = {"programmer:intu2tion"};
(programmer를 key 라고 한다. key : 가지고 오고 싶은 정보에 도달할 수 있는 열쇠!)
배열은 정보를 담는 그릇 + 순서대로 저장
객체는 정보를 담는 그릇 + 순서 없이 저장

함수를 통해 정리한 야간 모드
